Les pavages du fichier Cristallo.mac

Accueil

TeXgraph propose un fichier modèle Cristallo.mod écrit par Alphonse Capriani. Ce modèle permet le dessin d'un grand nombre de pavages du plan. Les pages qui suivent proposent une visualisation de ces pavages sous forme de vignettes. Un clic sur une vignette permet de voir l'image en grand au format pdf. Le fichier source est téléchargeable, à partir du lien sous la vignette. Attention: ces fichiers sources nécessitent le modèle Cristallo.mod que l'on trouvera ici sur le forum. Lire le fichier Cristallo.txt pour l'installation.
